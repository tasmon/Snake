# CloudSnake
A classic Snake game designed for CloudPhone, featuring T9 keypad support, multiple game modes, and customizable themes.

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/CloudSnake/`  
4. Upload icons if required.  
5. Save and refresh — **CloudSnake** will appear on your CloudPhone.
